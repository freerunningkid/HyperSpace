## ⚠️ 笔记添加铁律（2026-06-20 新增）

> 小金东明确要求：**严禁自行编造、补充、推断任何知识点。**

| 规则 | 说明 |
|------|------|
| **严格忠实原文** | 只记录小金东明确发送的内容，不添加任何他没说过的信息 |
| **不推测基数/费率** | 如果小金东只说了"冬季施工增加费"的基数，绝不能推断"雨季/夜间/特殊地区/行车干扰"也用同样基数 |
| **不补充遗漏项** | 如果资料中某项未提及（如其余6项措施费的基数），用"—"标注，绝不编造"同上" |
| **允许的加工** | 仅允许：格式化（表格/口诀/类比）、归类、添加记忆锚点、标注考点提示 |
| **不允许的加工** | 添加原文不存在的数据、推导未声明的规律、用自己的知识填补空白 |

> 💡 **核心原则**：笔记是"记录"不是"创作"。小金东没说的，就是不知道/还没确认的，宁可留空也不瞎编。
