# 一造备考资料整理

## 适用场景

小金东正在备考一级造价工程师。本 Skill 协助整理教材知识点、生成练习题、复盘错题、制定学习计划。

## 知识点整理模板

### 格式

```
## <科目名称> - <章节>

### 核心概念
- [概念名称]：[定义/公式]
- [概念名称]：[定义/公式]

### 公式卡片
| 公式 | 说明 | 适用场景 |
|------|------|---------|
| $公式$ | 变量含义 | 何时用 |

### 记忆口诀
[助记口诀]

### 易错点
- ⚠️ [常见错误1]
- ⚠️ [常见错误2]

### 真题练习
1. [题目]
   A. [选项A]
   B. [选项B]
   C. [选项C]
   D. [选项D]
   **答案**：[正确选项]
   **解析**：[详细解析]
```

## 学习计划模板

```
## <月份> 学习计划

### 本周目标
- [ ] [科目] 第X章 完成
- [ ] [科目] 真题卷 X 套
- [ ] 错题复盘 X 道

### 每日安排
| 时间 | 内容 | 备注 |
|------|------|------|
| 07:00-08:00 | [早读] | 公式/概念 |
| 20:00-22:00 | [精讲] | 教材+视频 |
| 22:00-22:30 | [刷题] | 章节练习 |
```

## 错题复盘模板

```
# 错题本

## <日期> 错题复盘

### 错题 1
- **题目来源**：[章节/真题]
- **错误答案**：[A/B/C/D]
- **正确答案**：[A/B/C/D]
- **错误原因**：[知识点不清/计算错误/审题失误]
- **知识点回顾**：[相关概念/公式]
- **下次注意**：[一句话提醒]
```
