# 工具编排速查

## 搜索与定位
- 文件内容 → grep
- 文件路径 → glob / ls
- 符号定义 → lsp_definition
- 代码架构 → codegraph_context
- 全盘搜索 → bash (Get-ChildItem)

## 编码修改
- 原子修改 → edit_file
- 批量修改同文件 → multi_edit
- 大段删除 → delete_range / delete_symbol
- 原子化编码任务 → run_skill("code-action")

## 验证闭环
- 语法检查 → lsp_diagnostics
- 运行测试 → bash 执行
- 验证 → run_skill("verify")
- 自动修复 → run_skill("auto-fix")

## 并行任务
- 子任务派发 → run_skill("dispatch")

## 文档处理
- PDF/Excel/Word → run_skill("document-tools")

## 钩子与规则
- 自动钩子 → run_skill("hooks-rules")

## 持久化
- 记住偏好 → remember()
- 删除错误记忆 → forget()
- 会话末写入 → run_skill("evolve")
