这套方案的“极低成本”主要来自三个方面：

2.  **免费API额度**：

### 🚀 分步操作指南

#### 第一阶段：本地部署 LobeHub

1.  **安装 Docker**：先在电脑上安装Docker。对于Windows或macOS用户，安装其桌面版应用即可。
2.  **执行一键脚本**：打开电脑终端（CMD或PowerShell），按顺序输入以下命令：
    *   创建并进入工作目录：`mkdir lobehub && cd lobehub`
    *   运行官方一键脚本：`bash <(curl -fsSL https://lobe.li/setup.sh) -l zh_CN`
3.  **选择部署模式**：脚本运行时，选择 **本地模式（选项1）**，这是最简单的“仅本机访问”模式。你需要确保本机 `3210`、`9000`、`9001` 端口没有被其他程序占用。
4.  **启动服务**：脚本配置完成后，运行命令启动：`docker compose up -d`
5.  **验证访问**：在浏览器中访问 `http://localhost:3210`，就能看到你的 LobeHub 界面了。

> **⚠️ 特别注意**：请务必使用官方推荐的一键脚本（`bash <(curl -fsSL https://lobe.li/setup.sh) -l zh_CN`）进行部署，这是最稳定且不受限制的方案。

#### 第二阶段：获取并配置 DeepSeek API

1.  **获取 API Key**：
    *   **注册获取**：前往 [DeepSeek 开放平台](https://platform.deepseek.com/)注册账号。
    *   **领取额度**：新用户会直接获得 **5亿 Tokens** 的免费额度。
    *   **创建密钥**：在平台的“**API keys**”页面创建新密钥，并**立即妥善保存**。
2.  **在 LobeHub 中配置**：
    *   在 LobeHub 界面进入 `设置` 页面。
    *   找到 `AI 服务商` 下的 `DeepSeek` 选项。
    *   开启服务，粘贴你复制的 API Key。
    *   选择 `deepseek-chat` 模型（即 V3 模型）开始对话。

### ⚠️ 可选：如何规避 R2 存储的潜在费用？

脚本默认使用 `Cloudflare R2` 作为文件存储。虽然免费额度足以覆盖个人日常使用，但要求**绑定支付方式**以完成验证，这可能会让一些人感到不便。

*   **方案一：绑定支付方式（推荐新手）**
    这是最简单、最省事的方案。只需绑定一个有效的支付方式，**仅用于身份验证，通常不会产生实际扣费**。

*   **方案二：改用本地存储（高级用户）**
    如果你希望完全不涉及外部服务，可以对 `docker-compose.yml` 文件进行自定义修改，将存储指向本地磁盘。
