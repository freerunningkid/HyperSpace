搞定了！Understand Anything 已就绪。

你可以直接在 Claude Code 里用这些命令了：

| 命令                           | 作用            |
| ---------------------------- | ------------- |
| `/understand`                | 分析当前项目，构建知识图谱 |
| `/understand-dashboard`      | 打开交互式图谱看板     |
| `/understand-chat <问题>`      | 直接问代码库的问题     |
| `/understand-explain <文件路径>` | 深度解析某个文件      |
| `/understand-diff`           | 分析改动影响范围      |
| `/understand-domain`         | 提取业务领域知识      |
| `/understand-onboard`        | 生成新手上手指南      |
| `/understand-knowledge <路径>` | 分析 LLM 知识库    |
