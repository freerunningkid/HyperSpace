# 场景速查

## 文件恢复（音乐/文档丢失）
1. 查回收站 → `Get-ChildItem '$Recycle.Bin' -Recurse -Filter *.mp3`
2. 全盘搜索 → `Get-ChildItem -Path D:\ -Include *.mp3,*.flac -Recurse`
3. 查清理工具日志
4. 汇总结果告诉小金东

## 新机器初始化
1. git clone 仓库
2. 启动 voice → voice_monitor.py
3. 检查 memory → 读 MEMORY.md

## Bug 排查
1. codegraph_context 获取代码架构
2. grep / lsp_definition 定位
3. code-action 修复
4. auto-fix 验证
5. debug 深度调查 → run_skill("debug")

## 文档处理
1. 判断文件类型（.pdf/.xlsx/.docx）
2. 调 document-tools 提取
3. 扫描件/图片 → screenshot-ocr 辅助
