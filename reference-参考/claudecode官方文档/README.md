# Claude Code 中文站

> 终端里的 AI 编程助手，一行命令装好就能用

## 特性

### 一行命令安装
macOS / Linux / WSL / Windows 都是一行命令搞定，国内镜像优先，无需翻墙、无需配置 key。

### 智能代码理解
深入理解你的代码库，分析项目结构，帮你快速定位问题和理解复杂逻辑。

### 自然语言交互
用中文描述需求，自动编写代码、修复 bug、重构或解释逻辑。

### 安全可控
文件操作和命令执行都需要你的确认，AI 行为始终在你掌控之中。

### 灵活的模型路由
主对话与子任务可以分别选模型，预算与效果两手抓；高级用户可改走官方 Anthropic / OpenRouter / 自有 provider。

### OpenClaw AI 管家
通过 OpenClaw 让 Claude 连接飞书、微信等平台，支持定时任务、文件管理、子代理。
